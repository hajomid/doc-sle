Part VII Synchronized Clocks with Precision Time Protocol

18 Precision Time Protocol

For network environments, it is vital to keep the computer and other devices' clocks synchronized and accurate. There are several solutions to achieve this, for example the widely used Network Time Protocol (NTP) described in Chapter 22, Time Synchronization with NTP.
